# 2.1.4 Character Key Shortcuts (Level A)

**Guideline**: 2.1 Keyboard Accessible  
**Principle**: 2 Operable

## Understanding

If a keyboard shortcut is implemented in content using only letter (including upper- and lower-case letters), punctuation, number, or symbol characters, then at least one of the following is true:

## Full Description

If a [keyboard shortcut](https://www.w3.org/TR/WCAG22/#dfn-keyboard-shortcuts) is implemented in content using only letter (including upper- and lower-case letters), punctuation, number, or symbol characters, then at least one of the following is true:

  

   
   **Turn off**: 
    A [mechanism](https://www.w3.org/TR/WCAG22/#dfn-mechanism) is available to turn the shortcut off;

      						
   **Remap**: 
     A mechanism is available to remap the shortcut to include one or more non-printable keyboard keys (e.g., Ctrl, Alt);

    **Active only on focus**: 
    The keyboard shortcut for a [user interface component](https://www.w3.org/TR/WCAG22/#dfn-user-interface-components) is only active when that component has focus.

## Details

- **Turn off**: A mechanism is available to turn the shortcut off;
- **Remap**: A mechanism is available to remap the shortcut to include one or more non-printable keyboard keys (e.g., Ctrl, Alt);
- **Active only on focus**: The keyboard shortcut for a user interface component is only active when that component has focus.

## Techniques

### Sufficient Techniques

**Providing a mechanism to allow users to remap or turn off character key shortcuts**


### Common Failures

- **F99**: Failure of Success Criterion 2.1.4 due to implementing character key shortcuts that cannot be turned off or remapped
