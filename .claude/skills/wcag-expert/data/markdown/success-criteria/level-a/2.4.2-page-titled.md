# 2.4.2 Page Titled (Level A)

**Guideline**: 2.4 Navigable  
**Principle**: 2 Operable

## Understanding

Web pages have titles that describe topic or purpose.

## Full Description

[Web pages](https://www.w3.org/TR/WCAG22/#dfn-web-page-s) have titles that describe topic or purpose.
