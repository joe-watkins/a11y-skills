# 3.1.1 Language of Page (Level A)

**Guideline**: 3.1 Readable  
**Principle**: 3 Understandable

## Understanding

The default human language of each web page can be programmatically determined.

## Full Description

The default [human language](https://www.w3.org/TR/WCAG22/#dfn-human-language-s) of each [web page](https://www.w3.org/TR/WCAG22/#dfn-web-page-s) can be [programmatically determined](https://www.w3.org/TR/WCAG22/#dfn-programmatically-determinable).

## Techniques

### Sufficient Techniques

**Using the language attribute on the HTML element**

**Setting the default language using the /Lang entry in the document catalog of a PDF document**

**Specifying the language for a passage or phrase with the Lang entry in PDF documents**


### Advisory Techniques

- **SVR5**: Specifying the default language in the HTTP header
