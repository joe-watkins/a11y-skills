# 1.3.2 Meaningful Sequence (Level A)

**Guideline**: 1.3 Adaptable  
**Principle**: 1 Perceivable

## Understanding

When the sequence in which content is presented affects its meaning, a correct reading sequence can be programmatically determined.

## Full Description

When the sequence in which content is presented affects its meaning, a [correct reading sequence](https://www.w3.org/TR/WCAG22/#dfn-correct-reading-sequence) can be [programmatically determined](https://www.w3.org/TR/WCAG22/#dfn-programmatically-determinable).

### Common Failures

- **F33**: Failure of Success Criterion 1.3.1 and 1.3.2 due to using white space characters to create multiple columns in plain text content

- **F49**: Failure of Success Criterion 1.3.2 due to using an HTML layout table that does not make sense when linearized
