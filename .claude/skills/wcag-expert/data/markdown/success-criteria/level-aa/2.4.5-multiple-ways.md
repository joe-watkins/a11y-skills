# 2.4.5 Multiple Ways (Level AA)

**Guideline**: 2.4 Navigable  
**Principle**: 2 Operable

## Understanding

More than one way is available to locate a web page within a set of web pages except where the web page is the result of, or a step in, a process.

## Full Description

More than one way is available to locate a [web page](https://www.w3.org/TR/WCAG22/#dfn-web-page-s) within a [set of web pages](https://www.w3.org/TR/WCAG22/#dfn-set-of-web-pages) except where the web page is the result of, or a step in, a [process](https://www.w3.org/TR/WCAG22/#dfn-processes).
