# 3.2.4 Consistent Identification (Level AA)

**Guideline**: 3.2 Predictable  
**Principle**: 3 Understandable

## Understanding

Components that have the same functionality within a set of web pages are identified consistently.

## Full Description

Components that have the [same functionality](https://www.w3.org/TR/WCAG22/#dfn-same-functionality) within a [set of web pages](https://www.w3.org/TR/WCAG22/#dfn-set-of-web-pages) are identified consistently.
