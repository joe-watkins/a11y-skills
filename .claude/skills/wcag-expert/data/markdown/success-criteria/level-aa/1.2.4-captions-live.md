# 1.2.4 Captions (Live) (Level AA)

**Guideline**: 1.2 Time-based Media  
**Principle**: 1 Perceivable

## Understanding

Captions are provided for all live audio content in synchronized media.

## Full Description

[Captions](https://www.w3.org/TR/WCAG22/#dfn-captions) are provided for all [live](https://www.w3.org/TR/WCAG22/#dfn-live)
      [audio](https://www.w3.org/TR/WCAG22/#dfn-audio) content in [synchronized media](https://www.w3.org/TR/WCAG22/#dfn-synchronized-media).
