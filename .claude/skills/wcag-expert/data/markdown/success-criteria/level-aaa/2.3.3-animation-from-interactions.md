# 2.3.3 Animation from Interactions (Level AAA)

**Guideline**: 2.3 Seizures and Physical Reactions  
**Principle**: 2 Operable

## Understanding

Motion animation triggered by interaction can be disabled, unless the animation is essential to the functionality or the information being conveyed.

## Full Description

[Motion animation](https://www.w3.org/TR/WCAG22/#dfn-motion-animation) triggered by interaction can be disabled, unless the animation is [essential](https://www.w3.org/TR/WCAG22/#dfn-essential) to the functionality or the information being conveyed.
