# 2.4.10 Section Headings (Level AAA)

**Guideline**: 2.4 Navigable  
**Principle**: 2 Operable

## Understanding

Section headings are used to organize the content.

## Full Description

[Section](https://www.w3.org/TR/WCAG22/#dfn-section) headings are used to organize the content.
   

   
   Note 1"Heading" is used in its general sense and includes titles and other ways to add a
      heading to different types of content.
   

   
   Note 2This success criterion covers sections within writing, not [user interface components](https://www.w3.org/TR/WCAG22/#dfn-user-interface-components). User interface components are covered under [Success Criterion 4.1.2](https://www.w3.org/TR/WCAG22/#name-role-value).
