# Process

series of user actions where each action is required in order to complete an activity

   
   Example 1Successful use of a series of web pages on a shopping site requires users to view
      alternative products, prices and offers, select products, submit an order, provide
      shipping information and provide payment information.
   

   
   Example 2An account registration page requires successful completion of a [Turing test](https://www.w3.org/TR/turingtest/) before
      the registration form can be accessed.
