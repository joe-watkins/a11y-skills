# Pointer Input

input from a device that can target a specific coordinate (or set of coordinates) on a screen,
      such as a mouse, pen, or touch contact
   

   NoteSee the [Pointer Events
      definition for "pointer"](https://www.w3.org/TR/pointerevents/#dfn-pointer) [[pointerevents](https://www.w3.org/TR/WCAG22/#bib-pointerevents)].
