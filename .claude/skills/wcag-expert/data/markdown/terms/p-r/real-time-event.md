# Real-Time Event

event that a) occurs at the same time as the viewing and b) is not completely generated
      by the content
   

   
   Example 1A Webcast of a live performance (occurs at the same time as the viewing and is not
      prerecorded).
   

   
   Example 2An on-line auction with people bidding (occurs at the same time as the viewing).

   
   Example 3Live humans interacting in a virtual world using avatars (is not completely generated
      by the content and occurs at the same time as the viewing).
