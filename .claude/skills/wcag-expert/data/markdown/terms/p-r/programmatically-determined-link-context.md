# Programmatically Determined Link Context

additional information that can be [programmatically determined](https://www.w3.org/TR/WCAG22/#dfn-programmatically-determinable) from [relationships](https://www.w3.org/TR/WCAG22/#dfn-relationships) with a link, combined with the link text, and presented to users in different modalities
   

   
   ExampleIn HTML, information that is programmatically determinable from a link in English
      includes text that is in the same [paragraph](https://html.spec.whatwg.org/multipage/dom.html#paragraph), 
      list item, or table cell as the link or in a table header cell that is associated with the table cell that contains the link.
   

   
   NoteSince screen readers interpret punctuation, they can also provide the context from
      the current sentence, when the focus is on a link in that sentence.
