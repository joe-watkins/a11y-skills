# Dragging Movement

an operation where the pointer engages with an element on the [down-event](https://www.w3.org/TR/WCAG22/#dfn-down-event) and the element (or a representation of its position) follows the pointer until an [up-event](https://www.w3.org/TR/WCAG22/#dfn-up-event)

   
   NoteExamples of draggable elements include list items, text elements, and images.
