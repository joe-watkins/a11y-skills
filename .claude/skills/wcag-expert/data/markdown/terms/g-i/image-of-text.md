# Image Of Text

text that has been rendered in a non-text form (e.g., an image) in order to achieve
      a particular visual effect
   

   
   NoteThis does not include text that is part of a picture that contains significant other visual content.
   

   
   ExampleA person's name on a nametag in a photograph.
