# Input Error

information provided by the user that is not accepted

   
   Note
     This includes:

   
     

      
      - Information that is required by the [web page](https://www.w3.org/TR/WCAG22/#dfn-web-page-s) but omitted by the user
      

      
      - Information that is provided by the user but that falls outside the required data
         format or values
