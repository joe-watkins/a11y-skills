# Synchronized Media

[audio](https://www.w3.org/TR/WCAG22/#dfn-audio) or [video](https://www.w3.org/TR/WCAG22/#dfn-video) synchronized with another format for presenting information and/or with time-based
      interactive components, unless the media is a [media alternative for text](https://www.w3.org/TR/WCAG22/#dfn-media-alternative-for-text) that is clearly labeled as such
