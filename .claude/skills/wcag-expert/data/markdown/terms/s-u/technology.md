# Technology

[mechanism](https://www.w3.org/TR/WCAG22/#dfn-mechanism) for encoding instructions to be rendered, played or executed by [user agents](https://www.w3.org/TR/WCAG22/#dfn-user-agents)

   
   Note 1As used in these guidelines "web technology" and the word "technology" (when used
      alone) both refer to web content technologies.
   

   
   Note 2Web content technologies may include markup languages, data formats, or programming
      languages that authors may use alone or in combination to create end-user experiences
      that range from static web pages to synchronized media presentations to dynamic Web
      applications.
   

   
   ExampleSome common examples of web content technologies include HTML, CSS, SVG, PNG, PDF,
      Flash, and JavaScript.
