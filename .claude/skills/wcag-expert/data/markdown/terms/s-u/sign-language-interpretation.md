# Sign Language Interpretation

translation of one language, generally a spoken language, into a [sign language](https://www.w3.org/TR/WCAG22/#dfn-sign-language)

   
   NoteTrue sign languages are independent languages that are unrelated to the spoken language(s)
      of the same country or region.
