# Label

[text](https://www.w3.org/TR/WCAG22/#dfn-text) or other component with a [text alternative](https://www.w3.org/TR/WCAG22/#dfn-text-alternative) that is presented to a user to identify a component within web [content](https://www.w3.org/TR/WCAG22/#dfn-content)

   
   Note 1A label is presented to all users whereas the [name](https://www.w3.org/TR/WCAG22/#dfn-name) may be hidden and only exposed by assistive technology. In many (but not all) cases
      the name and the label are the same.
   

   
   Note 2The term label is not limited to the label element in HTML.
